Website Deployment Tutorial

In this project, I deployed a Certificate of Grade System online using InfinityFree as the web hosting platform.
The website files were uploaded using FileZilla by connecting to the FTP server provided by the hosting account.
All project files such as HTML, CSS, JavaScript, and PHP were placed inside the htdocs directory to make them accessible online.
After uploading the files, I created a MySQL database through the InfinityFree control panel and used phpMyAdmin to set up the required tables for storing student records.
A PHP connection file was then configured to link the website to the database, allowing data from the Certificate of Grade system to be saved and retrieved properly.
Finally, I tested the live website using the provided domain to ensure that the system is working correctly, including form submission and database functionality.
This process successfully demonstrates how to deploy a fully functional web-based system online using FileZilla and PHP database integration.
