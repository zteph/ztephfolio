Website Deployment Tutorial

In this activity, I demonstrated how to manage and secure a website hosted on InfinityFree by exploring its control panel and security features.
Since InfinityFree does not provide full editable cPanel access like paid hosting, I used the available dashboard to locate important tools such as file management, domain settings, and database management.
For DNS configuration, I checked the assigned nameservers provided by InfinityFree, which are used to connect the domain to the hosting server. Although direct HTTPS/SSL configuration is limited, InfinityFree automatically provides free SSL support for subdomains, ensuring basic HTTPS protection for the website.
To secure data, I accessed phpMyAdmin through the control panel and performed database backup by exporting the SQL file, which can later be used for recovery if needed.
This export function serves as the main backup method for restoring the website’s database.
Overall, this process shows how DNS connection, SSL protection, and database backup can be managed even in a free hosting environment to help protect a personal web system.
